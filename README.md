# 🌐 Metroglam Studio - Responsive Website

A modern, fully responsive website built using **HTML, CSS, JavaScript, W3.CSS, and PHP**.  
This project includes multiple pages, a gallery system, a contact form with email functionality, and PWA support.

---

## 🚀 Features

- Fully responsive design (mobile, tablet, desktop)
- Clean and modern UI using W3.CSS
- Image gallery page
- Contact form with PHP email handling
- Form validation (client + server side)
- PWA support (manifest + icons)
- Cross-browser compatibility
- Lightweight and fast performance

---

## 🛠️ Technologies Used
```
- HTML5
- CSS3
- JavaScript
- W3.CSS Framework
- PHP (Mail Function)
- PWA (manifest.json, browserconfig.xml)
```

```

index.css → Custom styles

manifest.json → PWA configuration
browserconfig.xml → Windows tile configuration
```

## ▶️ How to Run This Project
```
Option 1: Simple HTML (Frontend only)
Clone repository
Open index.html in browser
```
---

```
Option 2: With PHP (Recommended)
Install XAMPP / WAMP
Move project to htdocs folder
Start Apache server
Open: http://localhost/your-project-folder/
```
